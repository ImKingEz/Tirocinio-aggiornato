Istruzioni per generare i locatori Robula+

1) Aprire la cartella robula-plus nel terminale
2) Esegui:
	python -m venv venv
	venv\Scripts\activate
	pip install .
3) Modifica script-generazione-locatori-robula.py inserendo il codice html e degli xpath dei tag su cui generare i locatori Robula+
4) Esegui con: python script-generazione-locatori-robula.py
